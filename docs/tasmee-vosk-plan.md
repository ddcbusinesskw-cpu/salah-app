# التسميع ثنائي المحرّك: Vosk (فوري) + whisper.cpp (دقيق) — الخطة

الهدف: توهّج فوري (أجزاء من الثانية) مع صوت المستخدم عبر Vosk streaming،
ودقّة قرآنية بالكلمات عبر whisper.cpp القرآني الذي يؤكّد عند نهاية كل آية —
أوفلاين بالكامل.

## الأصول

| الأصل | التفصيل |
|---|---|
| مكتبة Vosk | `com.alphacephei:vosk-android:0.3.75` (Maven Central، يُجلب وقت البناء) |
| موديل Vosk العربي | `vosk-model-ar-mgb2-0.4` · ~318MB · `https://alphacephei.com/vosk/models/vosk-model-ar-mgb2-0.4.zip` |
| موديل whisper | `Oa-4/whisper-tiny-ar-quran-gguf` (43MB، موجود) |
| الخطوط | QCF per-page (48MB، موجودة) |

**الاستضافة:** التطبيق ينزّل zip موديل Vosk مباشرة من alphacephei عند أول
استخدام (على الجهاز alphacephei متاح)، يفكّه على التخزين الداخلي، ويشير Vosk
إليه. لا يُحزَم في APK (يتجاوز حدّ Play 150MB).

**ملاحظة صوتية:** mgb2 مُدرَّب على العربية الإعلامية (MSA) لا التجويد؛ لذا
Vosk = فورية تقريبية تقود التوهّج، وwhisper القرآني = التأكيد النهائي الدقيق.

## المعمارية

```
AudioRecord 16kHz mono (الموجود في WhisperNativePlugin)
   ├─► Vosk Recognizer.acceptWaveForm(chunk)  ─► partial لحظي  ─► توهّج فوري (يقود)
   └─► نافذة whisper النامية (عند وقفة VAD)    ─► تأكيد قرآني   ─► تصحيح نهائي
```

Vosk يُغذّى من نفس خيط قراءة AudioRecord الحالي (لا مايك ثانٍ). whisper
يبقى كما هو (نافذة نامية + VAD). الدمج: Vosk يقود المؤشر لحظياً، whisper
يصحّح عند حدود الآية.

## المراحل

- **م٠ (هذه):** تبعية Vosk في gradle + الخطة + بصمة البناء.
- **م١:** plugin Vosk ستريمنغ — تنزيل/فكّ الموديل أصلياً (streaming، بلا تحميل
  318MB في ذاكرة JS)، `new Model`/`Recognizer`، تغذية من خيط AudioRecord،
  بثّ partials لحظية عبر أحداث الplugin.
- **م٢:** محاذاة partials مع كلمات الصفحة (snorm + تسامح تحرير) → المؤشر يتقدّم
  فوراً → توهّج لحظي + كشف خطأ الكلمة. الواجهة كما هي.
- **م٣:** عند وقفة/نهاية آية، whisper يؤكّد الآية ويصحّح التباس Vosk (دمج نظيف،
  whisper له الكلمة الأخيرة).
- **م٤:** تشخيص (المحرّكان، زمن Vosk، تأكيد whisper، النصوص) + قبول (توهّج خلال
  أجزاء من الثانية، دقة محفوظة، أوفلاين).

## القبول النهائي
